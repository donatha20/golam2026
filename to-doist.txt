📋 PHASE 1: IMMEDIATE SETUP TASKS
===============================================================================

□ 1.1 ENVIRONMENT SETUP
   □ Install Python 3.8+ on your system
   □ Install PostgreSQL 12+ database server
   □ Install Redis for caching (optional but recommended)
   □ Run: pip install -r requirements.txt
   □ Create .env file in project root (copy from .env.example)
   □ Set up virtual environment if not already done

□ 1.2 DATABASE CONFIGURATION
   □ Create PostgreSQL database: loan_management_db
   □ Create database user with appropriate permissions
   □ Update DATABASE_URL in .env file
   □ Run: python manage.py makemigrations
   □ Run: python manage.py migrate
   □ Create superuser: python manage.py createsuperuser
   □ Run: python manage.py collectstatic

□ 1.3 BASIC TESTING
   □ Test server startup: python manage.py runserver
   □ Test admin login at /admin/
   □ Test database connectivity
   □ Verify static files are serving correctly

📱 PHASE 2: SMS SERVICE SETUP (CHOOSE ONE OPTION)
===============================================================================

□ 2.1 OPTION A: TWILIO (RECOMMENDED - GLOBAL)
   □ Sign up at https://twilio.com
   □ Get Account SID from Twilio Console
   □ Get Auth Token from Twilio Console
   □ Purchase/get a Twilio phone number
   □ Add to .env file:
     SMS_ENABLED=True
     SMS_PROVIDER=twilio
     TWILIO_ACCOUNT_SID=your-account-sid
     TWILIO_AUTH_TOKEN=your-auth-token
     TWILIO_FROM_NUMBER=+1234567890
   □ Test SMS: python manage.py shell
     >>> from apps.core.sms_service import sms_service
     >>> sms_service.send_sms('+your-phone', 'Test message')

□ 2.2 OPTION B: TEXTLOCAL (INDIA FOCUSED)
   □ Sign up at https://textlocal.in
   □ Get API Key from dashboard
   □ Add to .env file:
     SMS_ENABLED=True
     SMS_PROVIDER=textlocal
     TEXTLOCAL_API_KEY=your-api-key
     TEXTLOCAL_SENDER=TXTLCL
   □ Test SMS functionality

□ 2.3 OPTION C: MSG91 (INDIA FOCUSED)
   □ Sign up at https://msg91.com
   □ Get Auth Key and Sender ID
   □ Add to .env file:
     SMS_ENABLED=True
     SMS_PROVIDER=msg91
     MSG91_AUTH_KEY=your-auth-key
     MSG91_SENDER_ID=MSGIND
   □ Test SMS functionality

□ 2.4 SMS TESTING
   □ Test payment reminders

⚙️ PHASE 5: AUTOMATED TASKS SETUP
===============================================================================

□ 5.1 CRON JOBS FOR SMS REMINDERS
   □ Open crontab: crontab -e
   □ Add daily payment reminders (9 AM):
     0 9 * * * /path/to/venv/bin/python /path/to/project/manage.py send_payment_reminders
   □ Add overdue reminders (10 AM):
     0 10 * * * /path/to/venv/bin/python /path/to/project/manage.py send_payment_reminders --overdue-only
   □ Add monthly interest calculation (1st of month, midnight):
     0 0 1 * * /path/to/venv/bin/python /path/to/project/manage.py calculate_monthly_interest
   □ Add daily database backup (2 AM):
     0 2 * * * pg_dump loan_management_db > /path/to/backups/db_$(date +\%Y\%m\%d).sql

□ 5.2 TEST AUTOMATED TASKS
   □ Test payment reminder command manually
