Environment Variables Configuration

Add these variables to your .env file to configure SMS services:

# Twilio SMS Configuration
SMS_ENABLED=True
SMS_PROVIDER=twilio
TWILIO_ACCOUNT_SID=your_account_sid_here
TWILIO_AUTH_TOKEN=your_auth_token_here
TWILIO_FROM_NUMBER=+1234567890

How to get credentials:
1. Sign up at twilio.com
2. Get your Account SID and Auth Token from the console
3. Purchase a phone number for sending SMS 

# TextLocal SMS Configuration
SMS_ENABLED=True
SMS_PROVIDER=textlocal
TEXTLOCAL_API_KEY=your_api_key_here
TEXTLOCAL_SENDER=TXTLCL

How to get credentials:
1. Sign up at textlocal.in
2. Get your API key from the settings
3. Configure your sender ID 

# MSG91 SMS Configuration
SMS_ENABLED=True
SMS_PROVIDER=msg91
MSG91_AUTH_KEY=your_auth_key_here
MSG91_SENDER_ID=MSGIND

How to get credentials:
1. Sign up at msg91.com
2. Get your Auth Key from the dashboard
3. Configure your sender ID

SMS Best Practices
Technical Guidelines
  • Keep messages under 160 characters for single SMS
  • Use UTF-8 encoding for international characters
  • Test thoroughly before bulk sending
  • Monitor delivery rates and costs
Content Guidelines
  • Always include your organization name
  • Use clear, professional language
  • Include relevant reference numbers
  • Respect timing (9 AM - 6 PM)
